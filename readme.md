### Project 2. ** Flipkart Console System (Python Project)**

This is a basic Flipkart-style console application developed using Python and Object-Oriented Programming (OOP) concepts.


### About the Project
This project includes the following features:
1. User Registration and Login
2. Product listing and selection
3. Category display
4. Order creation and order status update
5. Product review with rating

Each feature is created using separate classes like User, Product, Category, Order, and Review. The project shows how different classes work together in a small shopping system.

###  Concepts Used
Classes and Objects,
Instance variables,
Methods,
Dictionary data structure,
Date and time using datetime

###  Purpose of the Project
This project was created to understand OOP concepts in Python and to learn how an online shopping system works in a simple way.
